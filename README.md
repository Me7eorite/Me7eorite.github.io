## 📖 关于 M7 Lab

M7 Lab 是 [@me7eorite](https://github.com/Me7eorite) 的个人研究站，记录网络安全研究与技术学习的过程。

> 学技术与研究安全，是同一件事的两面。没有对底层的深刻理解，就没有真正的攻防能力。

## 🎯 内容方向

| 领域 | 内容 |
| :--- | :--- |
| ☕ **Java** | 反序列化链、中间件漏洞、Spring/Struts2 等框架研究、JVM 底层 |
| ⛓️ **Web3** | 智能合约审计、DeFi 攻击向量、Solidity / EVM |
| 🤖 **AI** | 提示注入、模型越狱、Agent 安全、LLM / RAG / 微调 |
| ☁️ **云原生** | K8s 攻防、容器逃逸、云配置审计、Docker / Serverless |
